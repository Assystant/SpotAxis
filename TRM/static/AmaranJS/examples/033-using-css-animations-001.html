AmaranJS effects not limited for built in ones you can combine with animate.css effects.Lets start series of examples with this one.

So if you want to use cool css effects first of all visit animate.css website and check effects and choose which one do you want to use.

Or, i created a tool for testing AmaranJS with Animate.css.Here is the tool.

$('#start').on('click',function(){
    $.amaran({
        'message'           :'My first example!',
        'cssanimationIn'    :'swing',
        'cssanimationOut'   :'bounceOut'
    });
});
